# Caesar Cipher

This project offers an easy-to-understand Python version of the Caesar Cipher. The project supports message encryption and decryption, preserves uppercase and lowercase letters, and includes input validation for shift values.

## Features
- Encrypt text with a custom shift
- Decrypt encrypted messages
- Preserve letter casing
- Validate shift values (1–25)

## Example

Input:
hello world

Encrypted (shift = 3):
khoor zruog
